<h2 style="color: #95a5a6;">Overview</h2>
<p>Welcome to <strong>Lio's Overhauled Villages</strong>! This mod is a transformation that brings a more modern and lively atmosphere to villages. Enjoy an immersive experience with upgraded structures, interactive elements, and hidden secrets waiting to be discovered in the libraries of each village.<br><br></p>
